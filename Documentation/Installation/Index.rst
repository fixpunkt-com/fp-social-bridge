============
Installation
============

The extension is installed via Composer:

..  code-block:: bash

    composer require fixpunkt/fp-social-bridge

Requirements
============

..  list-table::
    :header-rows: 1
    :widths: 30 70

    *   -   Component
        -   Version
    *   -   TYPO3
        -   **12.4 – 13.4**
    *   -   PHP
        -   **8.1 – 8.4**
